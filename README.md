# 🎁 Amigo Secreto

Aplicação web para sorteio de amigo secreto de forma simples e divertida.
Permite cadastrar participantes, realizar o sorteio aleatório e revelar 
os resultados ao final.

## 🚀 Funcionalidades

- Cadastro de participantes (mínimo 4 pessoas)
- Sorteio aleatório garantindo que ninguém tire a si mesmo
- Revelação de quem tirou quem ao final
- Opção de reiniciar o sorteio

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
